📦 dvnet-data.js
Principal

Contém todos os seus produtos, blog, trending e cupons. Você sobe esse arquivo no GitHub e o site atualiza sozinho — sem mexer no HTML.

📂 Páginas de Categoria
Novo

Gera uma página separada para cada categoria (headsets.html, monitores.html, etc.) com todos os produtos cadastrados e filtros por marca e preço.

💾 Backup dos dados

Salva todos os seus dados em JSON. Use para restaurar o admin em outro computador ou navegador.

🚀 Como publicar no site
1
Baixe o dvnet-data.js
Clique no botão ao lado. Um arquivo será baixado para o seu computador.
⚡ APENAS NA PRIMEIRA VEZ
2
Adicione 1 linha no index.html
Cole este código logo antes do </body> no seu index.html:
<script src="dvnet-data.js"></script>
Só faz isso uma vez. Da próxima, pula direto pro passo 3.
3
Sobe no GitHub
Arraste o arquivo dvnet-data.js para o seu repositório no GitHub. Se já existir, confirme a substituição.
✓
Pronto! Site atualizado
A Vercel detecta a mudança e publica em menos de 1 minuto. Seus produtos, preços e links já aparecem no site.
